/disable-tag
--------------
Delete tag by name

* Methods::

    POST 

* Authentication::

    Required.

* request parameters:

    - client:                    <application Name>. 

* Post Data:

    - s:     user/<usrId_>/label/<tag name>
    - t:     <tag name>
    - T:     //s3wAqZ_mEpOvpF8hGNO8Pw

* example::

    https://www.google.com/reader/api/0/disable-tag?client=scroll&hl=en

* response::

    OK
